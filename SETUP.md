# DCA Trading Bot — Фронтенд

## Что это такое

Веб-интерфейс для управления торговым DCA-ботом на Binance. Позволяет:

- Запускать и останавливать бота
- Настраивать параметры торговли (пара, бюджет, сетка и т.д.)
- Следить за статусом бота и открытыми ордерами в реальном времени

Фронтенд работает в браузере и общается с бэкендом через REST API.

---

## Стек

- **React 18 + TypeScript** — UI
- **Vite** — сборка и дев-сервер
- **Tailwind CSS + Radix UI** — стили и компоненты
- **TanStack Query** — запросы к API и polling статуса
- **Zustand** — состояние UI
- **Zod + React Hook Form** — валидация формы

---

## Как запустить

### 1. Установить зависимости

```bash
npm install
```

### 2. Создать файл `.env`

Скопируй `.env.example` в `.env` и укажи адрес твоего бэкенда:

```bash
cp .env.example .env
```

Содержимое `.env`:

```
VITE_API_BASE_URL=http://localhost:8000/api
```


### 3. Запустить в режиме разработки

```bash
npm run dev
```

Откроется на `http://localhost:5173`

### 4. Собрать для продакшена

```bash
npm run build
```

Результат окажется в папке `dist/` — её можно раздавать любым статическим сервером (nginx, Apache, Vercel и т.д.).

Для предпросмотра собранной версии локально:

```bash
npm run preview
```

## Что ожидается от бэкенда

Фронтенд обращается к трём эндпоинтам:

| Метод | Путь      | Описание                        |
|-------|-----------|---------------------------------|
| POST  | `/start`  | Запустить бота (тело — конфиг)  |
| POST  | `/stop`   | Остановить бота                 |
| GET   | `/status` | Получить текущий статус бота    |

---

## Важно

- **API Secret** не сохраняется в localStorage — нужно вводить каждый раз при запуске бота (сделано намеренно из соображений безопасности).
- Статус бота обновляется автоматически: каждые 2.5 сек когда бот работает, каждые 5 сек при ошибке.
